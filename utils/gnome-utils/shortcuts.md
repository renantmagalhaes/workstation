# Shortcuts

When using pop-shell, some default keybings must be changed.

## Lock Screen
```
Lock screen -> Super+L
```

## Window arrangement
```
View split on left -> Super+Left
View split on right -> Super+Right
Maximize Window -> Super+Up
Restore Window -> Super+Down
```

## switch windows

```
switch windows -> Alt+Tab
```
